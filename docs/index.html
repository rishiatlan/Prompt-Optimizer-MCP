How it works

You write a prompt. The MCP scores it, finds what's missing, and compiles a structured version. No AI calls inside — all intelligence comes from your host LLM.

1

Score your prompt

Get a quality score (0–100) across 5 dimensions: clarity, specificity, completeness, constraints, and efficiency. See exactly where it's weak.

2

Catch what's missing

10 deterministic rules detect ambiguities, missing constraints, scope creep, and blocking questions — before you waste a single API call.

3

Get a compiled prompt

Outputs a structured prompt with role, goal, constraints, workflow, and output format — compiled for Claude XML, OpenAI, or Markdown.

4

Know the cost before you run

Estimates token count and cost across 8 models from Anthropic, OpenAI, and Google. Recommends the best model for your task type.
